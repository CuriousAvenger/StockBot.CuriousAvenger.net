How to Choose Model Specification:

Email Address: Please enter your email address in order to recieve the results reguarding your stock information. Note: it may take around 1-2 hours for the model to train for your specified stock, so patience is much appreciated. The results should come as an email.
Company Ticker: A ticker symbol or stock symbol is an abbreviation used to identify publicly traded shares of a particular stock on a particular stock market. Examples being AMZN for Amazon, TSLA for Tesla, FB for Metaverse, MSFT for Microsoft
Perdict Into The Future: This is a required field, which inputs the number of days you want to predict into the future. So, if you input 1, then the model will predict 1 days ahead - tomorrow. However, the model's accuracy starts falling the more and more days you predict into the future.
Predict Using: This is an optional field - aka a hyperparameter - which decides how many days of information should the model use to predict given days into the future. Meaning, if you choose 1, then the model is going to use yesterdays data to predict into the future. Ideally the model defaults to 1, which is best for short terms results, but for long term accuracies, stick to 60.
